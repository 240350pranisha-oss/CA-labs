Title

:Design and Implementation of a Combinational Circuit Code Converter Using VHDL

Objective
:To study the concept of combinational circuits.
To understand different digital codes and code conversion.
To design a code converter using VHDL.
To simulate and verify the output using a VHDL simulator.

ntroduction

A code converter is a combinational logic circuit that converts data from one binary code to another without using memory elements. Since the output depends only on the present input, code converters are classified as combinational circuits.

Code converters are widely used in digital electronics, communication systems, microprocessors, digital displays, calculators, and embedded systems. They simplify data representation and improve compatibility between different digital devices.

In this experiment, a Binary to Gray Code Converter is designed using VHDL. Gray code is commonly used because only one bit changes between consecutive numbers, reducing errors during transitions.
