TITLE

Design and Implementation of a Combinational Circuit Code Converter Using VHDL

Objective
TO study the concept of combinational circuits. To understand different digital codes and code conversion. To design a code converter using VHDL. To simulate and verify the output using a VHDL simulator.

INTRODUCTION:

A code converter is a combinational logic circuit that converts data from one binary code to another without using memory elements. Since the output depends only on the present input, code converters are classified as combinational circuits.Code converters are widely used in digital electronics, communication systems, microprocessors, digital displays, calculators, and embedded systems. They simplify data representation and improve compatibility between different digital devices. In this experiment, a Binary to Gray Code Converter is designed using VHDL. Gray code is commonly used because only one bit changes between consecutive numbers, reducing errors during transitions.

THEORY:

A combinational circuit is a digital circuit in which the output depends only on the current combination of input values. It contains no memory or feedback.

Characteristics

No storage elements. Output depends only on present inputs. Fast operation. Easy to design using Boolean expressions. Used in adders, multiplexers, encoders, decoders, comparators, and code converters.

DISCUSSION:

The Binary to Gray code converter was successfully implemented using VHDL. The circuit behaved as a combinational circuit because the output depended only on the present input values. Simulation verified that the conversion equations produced correct Gray code outputs for every binary input combination. This experiment demonstrated the practical use of combinational logic design and VHDL modeling in digital systems.

CONCLUSION:

The Binary to Gray Code Converter was successfully designed, implemented, and simulated using VHDL. The obtained outputs matched the theoretical results for all input combinations. This experiment improved the understanding of combinational circuits, code conversion techniques, Boolean logic, and hardware description using VHDL. The experiment also demonstrated the importance of code converters in modern digital electronics and computer engineering.
